# different ways to give css  text colors:
1. named color like red, green, blue, yellow etc
2. RGB color --->(red, green, blue) (0 to 255, 0 to 255, 0 to 255)
3. HSL color ---> (hue, saturation, lightness) 
    - hue is color type basically color from color wheel (0 to 360)
    - saturation: intensity of color,,,, 0% greyscale ,,, 100% fully saturated (in % 0 to 100%)
    - lightness  in %(opacity 0%: black, 100%: white)
4. hexadecimal(0 to f in 8 bit rgba or rgb)
5. rgba (also hsla possible)
    - a: alpha (intensity / transparency)---> value in between 0 to 1 
    - 0: white, transparent,  1: opaque
